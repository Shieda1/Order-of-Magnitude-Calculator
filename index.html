What is the order of magnitude?
The order of magnitude is an approximation of the logarithm of a value relative to some contextually understood reference value, usually 10, interpreted as the base of the logarithm and the representative of values of magnitude one. How to calculate order of magnitude?
To calculate order of magnitude you should use the following formula:
N = a * 10^b
where:
N = order of magnitude of a number
a = integer a
b = order of magnitude of the number
Why use order of magnitude?
The order of magnitude is used in order to make approximate comparisons. If numbers differ by one order of magnitude then x is about ten times different in quantity than y. If values differ by two orders of magnitude then they differ by a factor of about 100.
